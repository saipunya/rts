Put Thai font files here for Dompdf to embed.

Required files:
- THSarabunNew.ttf
- THSarabunNew-Bold.ttf

After copying these .ttf files, per-row PDF will render Thai correctly.
You can use "TH Sarabun New" from standard Thai fonts or any Thai-capable font
(rename in export_rubber_pdf.php if you use different names).
